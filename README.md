# 🚀 Streamlit Framework Project

A simple web application built using Python and Streamlit to demonstrate how quickly interactive web apps can be created without writing HTML, CSS, or JavaScript.

---

## 📌 About the Project

This project is a beginner-friendly Streamlit application that shows how Python scripts can be converted into interactive web applications.

It is useful for:
- Data Science projects
- Machine Learning model demos
- Dashboards
- Quick web prototypes

---

## 🛠 Tech Stack

- Python 3.x
- Streamlit
- Pandas
- NumPy
